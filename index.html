Hungarian Job Market Analysis

This project analyzes the current state of the data analyst job market in Hungary by scraping, cleaning and analyzing over 100 job postings from profession.hu. The goal was to identify the most in-demand skills, experience requirements and geographic trends to help aspiring data analysts make informed career decisions.

Using Python and BeautifulSoup, I built a custom web scraper that collected job listings and their full descriptions across multiple search terms, handling deduplication and data quality automatically. The data was then analyzed with pandas to extract technical skills, experience levels and location patterns.

Key findings include SQL and Python as the most demanded skills in the Hungarian market, Power BI dominating as the BI tool of choice and Budapest accounting for 40% of all data analyst positions. Visualizations were created using matplotlib and seaborn, including a skill co-occurrence heatmap showing which skills employers expect together.

  • Tech: Python, BeautifulSoup, pandas, matplotlib, seaborn
  • Data: 100+ job postings scraped from profession.hu
  • Output: Skill demand analysis, experience level breakdown, geographic insights

Montgomery County Sales Analysis

A business intelligence project analyzing 307,000+ liquor sales transactions from Montgomery County, Maryland (2017–2020). The goal was to identify sales trends, optimize inventory management and provide strategic recommendations, simulating the role of a Data Analyst working for a real organization.

Using Python and pandas, I cleaned and transformed the raw dataset, engineered new features and performed multi-dimensional analysis covering product performance, supplier concentration, channel distribution and seasonal patterns.

Key findings include Beer accounting for 70% of total sales, July being the peak sales month, the warehouse channel driving 78% of volume and the top 5 suppliers representing 60% of sales. Results were communicated through an executive-level dashboard built with matplotlib and seaborn.

  • Tech: Python, pandas, matplotlib, seaborn, NumPy
  • Data: 307,645 records from Montgomery County open data
  • Output: Executive dashboard, trend analysis, business recommendations

Data Cleaning & Visualization Project

This project explores over 17,000 historical restaurant menus collected between 1840 and 2015. I was curious to see how food culture evolved over time but before analyzing anything, the data needed serious cleaning. Using SQL (PostgreSQL), I handled missing values, fixed inconsistent date formats, standardized text fields and converted currency symbols into ISO codes to make the dataset reliable and ready for analysis.

After cleaning, I built an interactive dashboard in Power BI that visualizes how menus changed over time with clear spikes in the early 1900s. It also highlights the most common occasions like daily, anniversary and complimentary events and shows how different venue types (commercial, social and government) contributed to the data. A custom date filter and summary cards make it easy to explore specific periods and discover new insights.

This project was a great reminder of how much value comes from cleaning and structuring data, the foundation for any meaningful analysis or storytelling.
