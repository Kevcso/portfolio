Data Cleaning & Visualization Project

This project explores over 17,000 historical restaurant menus collected between 1840 and 2015. I was curious to see how food culture evolved over time but before analyzing anything, the data needed serious cleaning. Using SQL (PostgreSQL), I handled missing values, fixed inconsistent date formats, standardized text fields and converted currency symbols into ISO codes to make the dataset reliable and ready for analysis.

After cleaning, I built an interactive dashboard in Power BI that visualizes how menus changed over time with clear spikes in the early 1900s. It also highlights the most common occasions like daily, anniversary and complimentary events and shows how different venue types (commercial, social and government) contributed to the data. A custom date filter and summary cards make it easy to explore specific periods and discover new insights.

This project was a great reminder of how much value comes from cleaning and structuring data, the foundation for any meaningful analysis or storytelling.
